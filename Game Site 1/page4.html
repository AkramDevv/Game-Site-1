Tekken 8

About the Game

Tekken 8 is the latest installment in the iconic fighting game series developed by Bandai Namco Entertainment. The game features stunning graphics, fluid animations, and a diverse roster of characters. Players can engage in intense battles with friends or compete against others online. With new mechanics and enhanced gameplay, Tekken 8 promises to deliver an unforgettable fighting game experience.

System Requirements

  • Minimum:
    • OS: Windows 10 (64-bit)
    • Processor: Intel Core i5-7500 / AMD Ryzen 5 1600
    • Memory: 8 GB RAM
    • Graphics: GeForce GTX 1060 / Radeon RX 580
    • Storage: 60 GB available space
  • Recommended:
    • OS: Windows 10 (64-bit)
    • Processor: Intel Core i7-7700 / AMD Ryzen 7 2700
    • Memory: 16 GB RAM
    • Graphics: GeForce RTX 2070 / Radeon RX 5700 XT
    • Storage: 60 GB available space
